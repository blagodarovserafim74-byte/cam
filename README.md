# IP Camera Viewer

Проект на Python для подключения к IP-камере и показа видеопотока в окне программы.

## Что изменено

Структура сделана более жизненной и пригодной для развития:

- логика камеры вынесена в `services/`;
- интерфейс вынесен в `ui/`;
- настройки вынесены в `config.py`;
- проверки и преобразования вынесены в `utils/`;
- сообщения между потоками описаны отдельно в `models.py`.

## Новая возможность

Теперь прямо в программе можно **найти рабочую ссылку потока** для своей камеры.

Это работает так:

1. В блоке **«Поиск ссылки прямо в программе»** вводится:
   - IP или адрес камеры;
   - логин;
   - пароль.
2. Нажимается кнопка **«Найти ссылку»**.
3. Программа проверяет типовые RTSP и HTTP-ссылки **только для указанного хоста**.
4. Если ссылка найдена, она автоматически вставляется в поле подключения.

Важно:
- программа не сканирует всю сеть;
- программа не подбирает пароли;
- проверяются только стандартные варианты ссылок для указанной камеры.

## FFmpeg

Теперь видеопоток открывается через **FFmpeg**.

Что нужно для работы:
- установить FFmpeg;
- чтобы `ffmpeg` и `ffprobe` были доступны в `PATH`.

Проверка в терминале:

```bash
ffmpeg -version
ffprobe -version
```

Если команды не находятся, добавь папку `bin` из FFmpeg в системный `PATH`.
