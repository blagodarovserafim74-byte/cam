# IP Camera Viewer

## Структура проекта

```text
cam/
├── README.md          # Описание проекта, запуск, примеры ссылок и пояснения по интерфейсу
├── requirements.txt   # Зависимости Python
└── src/
    └── main.py        # Полный исходный код приложения
```

## Какие файлы нужны

- `src/main.py` — основная программа с графическим интерфейсом.
- `requirements.txt` — список библиотек для установки.
- `README.md` — инструкция по запуску и использованию.

## Какие модули используются

- `tkinter` — создание графического интерфейса.
- `threading` — чтение видеопотока в отдельном потоке без зависания окна.
- `queue` — передача кадров и статусов между потоком камеры и GUI.
- `dataclasses` — описание структуры сообщений между потоками.
- `cv2` (`OpenCV`) — подключение к IP-камере и чтение кадров.
- `PIL.Image` и `PIL.ImageTk` (`Pillow`) — показ кадров OpenCV в окне Tkinter.

## За что отвечает каждая часть программы

- `FrameMessage` — контейнер для кадра, статуса подключения или текста ошибки.
- `CameraWorker` — фоновый поток для подключения к камере и чтения видео.
- `IPCameraApp` — основной класс приложения:
  - рисует красивую рабочую область в тёмной теме;
  - создаёт левую панель управления;
  - создаёт правую область просмотра видео;
  - обновляет цветной статус подключения;
  - корректно останавливает поток без зависаний.

## Что есть в интерфейсе

- поле для ввода ссылки на IP-камеру;
- кнопка **«Подключиться»**;
- кнопка **«Отключиться»**;
- большая рабочая область для показа видео;
- цветной блок статуса подключения;
- карточка с примерами ссылок;
- карточка с краткими подсказками.

## Куда вставлять ссылку на камеру

После запуска программы ссылка вставляется в левую панель, в поле **«URL камеры»**.

После этого нажмите кнопку **«Подключиться»**.

## Примеры ссылок

### RTSP

```text
rtsp://login:password@192.168.1.10:554/stream
```

### HTTP / MJPEG

```text
http://192.168.1.10:8080/video
```

## Инструкция по запуску на Windows

### 1. Установите Python

Рекомендуется Python 3.11 или новее.
Во время установки на Windows включите опцию **Add Python to PATH**.

### 2. Установите зависимости

Откройте `cmd` или PowerShell в папке проекта и выполните:

```bash
pip install -r requirements.txt
```

### 3. Запустите программу

Из корня проекта выполните:

```bash
python src/main.py
```

## Как работает программа

1. Пользователь вставляет URL потока в поле `URL камеры`.
2. Нажимает **«Подключиться»**.
3. Отдельный поток `CameraWorker` открывает камеру через OpenCV.
4. Кадры передаются в GUI через `queue.Queue`.
5. Интерфейс показывает видео в правой рабочей области.
6. При ошибке выводится понятное сообщение в статусе.
7. При нажатии **«Отключиться»** поток останавливается, а видео прекращается без зависаний.
